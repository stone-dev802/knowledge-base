# Scrum

## Qu'est-ce que Scrum ?
Scrum est une méthode Agile utilisée pour gérer les projets informatiques.

## Sprint
Un sprint est une période courte (1 à 4 semaines) pendant laquelle l'équipe réalise des tâches.

## Rôles dans Scrum
- Product Owner : définit les besoins
- Scrum Master : aide l'équipe à suivre Scrum
- Développeurs : réalisent le travail

## Événements Scrum
- Sprint Planning
- Daily Scrum
- Sprint Review
- Sprint Retrospective

## Artefacts
- Product Backlog
- Sprint Backlog
- Increment
